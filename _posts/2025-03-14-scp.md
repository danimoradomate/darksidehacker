Linux es un estuche de monerias bastante utiles, como SCP en Linux, que es una herramienta de línea de comandos utilizada para transferir archivos de forma segura entre sistemas locales y remotos, o entre dos sistemas remotos. SCP utiliza el protocolo SFTP sobre SSH para la transferencia de datos, lo que garantiza la seguridad y la autenticación de la conexión.

````markdown
scp [[destino1@]hostname1:]file1 ... [[destino2@]hostname2:]file2

````

Un ejemplo tangible es:

1. Copiar un archivo de un sistema remoto a un directorio local específico:

```c++
scp /ruta/de/archivo_local/my_file.txt usuario@user_remote:/ruta/de/destino
````

2.- Copiar un archivo de un directorio local a un sistema remoto específico:

```c++
scp -r /ruta/de/archivo_local/my_directory usuario@user_remote:/ruta/de/destino
```

3.- Copiar un directorio completo de un sistema remoto a un directorio local específico:

```c++
scp -r usuario@sistema_remoto:/ruta/de/directorio_remoto /ruta/de/destino/local/
```

```c++
scp usuario@sistema_remoto:/ruta/de/origen/archivo_remoto .
```

El el segundo "." al final de la línea de comandos hace referencia al directorio actual en el sistema local.

4.- Copiar un archivo remoto a otro sistema remoto:

```c++
scp -3 usuario1@sistema_remoto1:/ruta/de/archivo_remoto usuario2@sistema_remoto2:/ruta/de/destino/remoto/
```

Si se requieren mas opciones como tipo de cifrado de datos, puerto de contexión consulta el siguiente <a href="https://man7.org/linux/man-pages/man1/scp.1.html">enlace.</a>
````
